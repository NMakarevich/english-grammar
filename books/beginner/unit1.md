5. Grammar Reference

5.1 am/are/is

|         |                    |              |
|---------|--------------------|--------------|
| I       | **'m**<br>**am**   | Serena       |
| You     | **'re**<br>**are** | Tom          |
| My name | **'s**<br>**is**   | James Bond   |
| This    | **is**             | Paul Bartozs |

5.2 Question with question words

**What's** your name?
**What's** this in English?
**How** are you?

5.3 Possessive adjectives

**My** name's John.
What's **your** name?

5.4 a/an

We use _an_ before words that begin with _a, e, i, o_ or _u_.

**an** apple

**an** umbrella

**an** English book

but:

**a** bike

**a** phone

**a** house

5.5 Plural nouns

1. Most nouns add _-s_.

book -> book**s**

phone -> phone**s**

laptop -> laptop**s**

2. Some nouns add _-es_

sandwich -> sandwich**es**

bus -> bus**es**

watch -> watch**es**
