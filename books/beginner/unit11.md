## 1. Grammar Spot

**_Can/can't_** have the same form for all persons. <br>
I / You / He / She / It / We / They **can/can't** speak French. <br>
He **can** make cakes. He **can't** run fast. <br>
**Can** you cook? Yes, I **can**.

## 2. Grammar Spot

1. Adverbs can come after the verb. <br>
They can run **fast**. <br>
He plays football **well**
2. Regular adverbs end in **_-ly_** <br>
She can speak French **fluently**. <br>
Please speak more **slowly**.

## 3. Grammar reference

### 3.1 can

**Positive**

|                                                 |         |                                                |
|-------------------------------------------------|---------|------------------------------------------------|
| I <br/> You <br/> He/She/It <br/> We <br/> They | **can** | swim. <br/> drive. <br/> cook. <br/> run fast. |

**Negative**

|                                                 |           |                                            |
|-------------------------------------------------|-----------|--------------------------------------------|
| I <br/> You <br/> He/She/It <br/> We <br/> They | **can't** | draw. <br/> speak German. <br/> play golf. |

**Yes/No questions and short answers**

|                         |                   |
|-------------------------|-------------------|
| **Can** you swim?       | Yes, I **can**.   |
| **Can** he play tennis? | No, he **can't**. |

We don't use _do/does/don't/doesn't_ with _can_.

I **can't** swim. NOT ~~I don't can swim~~ <br>
**Can** you cook? NOT ~~Do you can cook?~~ <br>
She **can't** speak Spanish. NOT ~~She doesn't can...~~ <br>
They **can't** dance. NOT ~~They don't dance~~

### 3.2 Adverbs

1. Adverbs give more information about verbs.
go **fast** <br>
draw **well** <br>
sing **beautifully**
2. Notice the word order.
You **speak** English **well**. NOT ~~You speak well English.~~ <br>
He **drives** his car **fast**. NOT ~~He drives fast his car.~~
3. Regular adverbs end in **_-ly_**

| Adjective | Adverb      |
|-----------|-------------|
| fluent    | fluently    |
| beautiful | beautifully |
| slow      | slowly      |
| careful   | carefully   |
| usual     | usually     |

4. Some adverbs are irregular

| Adjective | Adverb |
|-----------|--------|
| good      | well   |
| fast      | fast   |
| late      | late   |
| early     | early  |
| hard      | hard   |
